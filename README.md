# busy-honeys

모의 주식 거래 사이트 개발 저장소 입니다.

## 목적

- 재테크에 대한 지식이 없는 사회 초년생 / 직장인을 대상으로 재테크 중 하나 인 주식에 대해 알려주는 모의 주식 거래 사이트를 개발한다.
- 주식 거래 및 차트를 보는 법을 주 컨텐츠로 삼아 제공한다.
